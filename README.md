J'ai réalisé ce projet avec un camarade lorsque j'étais en classe de première au lycée dans le cadre du cours de NSI. 
Il vise à créer un site sur le thème de l'anime Dr stone en utilisant notamment les langages html, css et javascript,
pour le corps du site, une mise en page propre et structurée et diverses animations originales.

Pour visualiser correctement le site web, il est préférable d'utiliser un navigateur comme Chrome.
Il faut télécharger tout le dossier et éxécuter le fichier dr_stone.html, n'oubliez pas d'activer le son pour plus d'immersion ;)
